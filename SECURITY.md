# Security Policy

We take the security of the **NagarRakshak** platform seriously. Thank you for helping us keep our community reports and citizen profiles secure.

---

## Supported Versions

Only the latest release version of the platform is actively supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |
| < 1.0   | :x:                |

---

## Reporting a Vulnerability

**Please do not open a public GitHub issue for security vulnerabilities.**

If you discover a security vulnerability, please report it privately:
1. Email us at `security@nagarrakshak.org`.
2. Provide details of the vulnerability, including:
   - Affected components (Android app client, Laravel API backend, SQLite configuration).
   - A description of the exploit vectors and impact.
   - Proof of concept (PoC) steps or script to reproduce the issue.

We will acknowledge receipt of your report within **48 hours** and provide a timeline for coordination and fix deployment.
